- Euclidean Distance-Based Matching

- Extract metrics from each detected tree crown (coordinates, crown radius, area)

- Calculate the Euclidean distance between crowns

- Link each tree crown to its closest match according to this distance metric

## Slide 7

- Euclidean Distance-Based Matching

- This approach has been implemented for aligning field-surveyed trees with detected individual tree crowns, https://pmc.ncbi.nlm.nih.gov/articles/PMC6330952/

## Slide 8

- IoU-Based Overlap Matching

- Compare each detected tree crown in different orthomosaics and measure how much they overlap using Intersection over Union (IoU).

- How It Works:

- Compute the IoU score between crowns from different orthomosaics.

- If IoU > threshold (e.g., 0.75), the crowns are considered the same tree.

- Assign the same tree ID to matching crowns.
