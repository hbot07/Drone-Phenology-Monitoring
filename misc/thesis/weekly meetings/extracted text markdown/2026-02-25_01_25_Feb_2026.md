## Slide 17

- Clustering based on Phenology Signals

- For each of the 20 fully-tracked trees, we extract 5 metrics across 7 OMs: gcc, rcc, veg_fraction, gray_std_smooth, gray_entropy

- Each tree's time series for each metric is rescaled to [0, 1] independently

- The 5 normalised series (each 7 points) are concatenated into a single 35-element feature vector per tree

- scipy.cluster.hierarchy.linkage(X, method='ward') builds a dendrogram(hierarchical/agglomerative clustering)

- Ward is robust to the small sample size. It produces compact, similarly-sized clusters

## Slide 18

- Clustering Method: Ward Linkage

- Each feature column is z-score normalized (mean=0, std=1) so that all metrics have equal weight

- Distance Computation: Euclidean distance on the standardized 63-dimensional vectors

- This treats the 7-OM time series as one concatenated feature vector

- So differences in both magnitude AND timing contribute to distance
