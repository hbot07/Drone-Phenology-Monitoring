- How do we match trees across runs?

- Using their geospatial coordinates

- Can use scikit-learn’s Nearest Neighbors

- find the closest previous detection for each new detection

- What about misaligned images?

- detect keypoints in both images (using algorithms like ORB, SIFT, or SURF), match those keypoints, compute a transformation (homography or affine transformation) using RANSAC, and then warp one image to align with the other.
