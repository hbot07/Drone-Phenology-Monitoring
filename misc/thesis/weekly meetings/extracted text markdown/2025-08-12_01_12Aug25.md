## Slide 3

- Workstation Status

- WSL stores Ubuntu in a special virtual disk format, so it took 2-3 days to figure out how to recover files from Ubuntu.

- Since Disk C in the workstation is full, nothing can be stored there, so we had to choose another drive for storing files, which also took time.

- Decided to use TeamViewer instead of SSH.

- Downloads are stored in Disk C by default, so we needed to change the default download location to download TeamViewer.

- VS Code generally stores its extensions in Disk C, so we need to change that location as well and then everything will work fine
