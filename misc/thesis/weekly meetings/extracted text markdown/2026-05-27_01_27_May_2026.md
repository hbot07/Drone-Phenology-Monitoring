## Slide 11

- Features

- Indices

- NDVI = (B8 - B4) / (B8 + B4)

- GNDVI = (B8 - B3) / (B8 + B3)

- NDRE = (B8A - B5) / (B8A + B5)

- NDMI = (B8 - B11) / (B8 + B11)

- NBR = (B8 - B12) / (B8 + B12)

- EVI

- visible greenness: B3 / (B2 + B3 + B4)

- redness: B4 / (B2 + B3 + B4)

## Slide 13

- Metrics

- accuracy = (TP + TN) / (TP + TN + FP + FN)

- balanced accuracy = average of recall for each class

- Acacia recall = correctly detected Acacia / total actual Acacia

- Non-acacia recall = correctly detected Non-acacia / total actual Non-acacia

- balanced accuracy = (Acacia recall + Non-acacia recall) / 2

- Precision, Recall, F1 Score
