## Slide 3

- Split Case Logic

- Identify all one-to-many nodes (split nodes).

- For each split node:

- -Compute similarity scores for all successors.

- -Select top 3 similarity scores (or fewer if less than 3 successors).

- From these top candidates:

- -Compute similarity with Their front nodes.

- -Choose the successor with the highest forward similarity.

- Special handling:

- -If the split occurs at the second-last node,

- then directly select the successor with the highest similarity at that stage.

## Slide 4

- Merge Case Logic

- For merge (many-to-one) cases:

- -Compare similarity of all merging candidates with the back node.

- -Select the candidate with the highest similarity score.

- If a merge is followed by a split again:

- -Apply the same split logic recursively.

## Slide 5

- Containment-Based Rules

- Prefer successors that:

- -Maximise contained geometry

- -Maintain good area balance

- Use gates to filter candidates:

- -IoU threshold

- -Centroid proximity

- Tie-breaking rules:

- -overlap_prev (if previous crown contains current)

- -overlap_curr (if current crown contains previous)

- -balanced_area_ratio

## Slide 6

- Shedding Event Detection

- When do we detect a shedding event?

- Previous crown polygon contains the current polygon (prev_contains_curr = True)

- High overlap of current polygon inside previous

- overlap_curr ≥ 0.85

- Large drop in area compared to previous

balanced_area_ratio ≤ 0.6
 (current crown significantly smaller than before)

- What to do when shedding is detected?

- Prefer the smaller contained polygon as the true successor

- It is well-aligned with the previous crown

- Likely a partial detection rather than a new crown

- Prevents incorrect matching to a nearby larger crown

## Slide 7

- Partial-Overlap Case

- Treat partial-overlap successors as soft one-to-one matches.

- From all partial-overlap successors:

- -Select top-K based on similarity.

- Final successor chosen using a continuity score combining:

- -IoU

- -Overlap fractions

- -Base similarity

- -Centroid proximity
